# DADA2
## Codes for DADA2


### Installation
```
if (!require("BiocManager", quietly = TRUE))
  install.packages("BiocManager")
BiocManager::install("dada2")
```

- `DADA2.R`: Bacterial 16S rRNA gene amplicon sequencing analysis.
